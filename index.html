Présentation

Ce miniprojet a pour but de préparer le projet de groupe et vise à renvoyer un tableau avec les informations des pages internet d'un fichier d'URLs.

Vous pouvez consulter ce tableau pour visualiser les informations récupérées. Le tableau apporte les informations suivantes :

  • Numéro de l'URL
  • URL concernée
  • Code HTTP de la requête
  • Encodage
  • Nombre de mots
